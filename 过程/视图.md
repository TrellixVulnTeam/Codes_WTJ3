| eat_int_store         | CHAR     | 选填 (u'T', u'堂食'), (u'W', u'外送')                        |
| specify_delivery_time | DATETIME | 选填                                                         |
| location              | INT      | 必填                                                         |
| payment_method        | CHAR     | 选填，(u'W', u'微信支付'), (u'Z', u'支付宝'), (u'Y', u'余额支付') |

| payment_account_id    | INT      | 必填                                                         |
